O que é flask?


Basicamente o flask é um micro framework, que vai utilizar a linguagem python para conseguir criar aplicações de web. Podemos destacar que o flask é muito poderoso e ele geralmente é ideal para quem busca: Rapidez, simplicidade, soluções para projetos pequenos, e para aplicações robustas.
